# Dad-Jokes-Generator-II

Dad Jokes Generator II — a small, friendly web app built with HTML, CSS, and JavaScript that serves up classic dad jokes at the click of a button. Perfect for practicing frontend basics or adding a little humor to your day.

## Features

- Generates a random dad joke on demand
- Clean, responsive UI using HTML and CSS
- Lightweight: no build step or backend required — just open in your browser

## Demo / Usage

1. Clone this repository:

   git clone https://github.com/BinaryVortex/Dad-Jokes-Generator-II.git

2. Open `index.html` in your browser (double-click or right-click -> Open with -> your browser).

3. Click the "Get Joke" button (or whatever the button in the app is labeled) to display a new dad joke.

Note: If the project uses a different filename for the main HTML (for example `index.html` or `app.html`), open that file instead.

## Development

- Languages: HTML, CSS, JavaScript
- No dependencies or package managers required. Edit the HTML/CSS/JS files directly and refresh the browser to see changes.
